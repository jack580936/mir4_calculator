# mir4_calculator

這是一個簡單的計算器，能夠提供Mir4的玩家一些能讓遊戲更容易遊玩的小功能。

您可以從此進入網站： https://jack580936.github.io/mir4_calculator/

* 裝備材料計算
* 體質材料計算
* 奇緣攻略
* 各地圖資源分布
* 委託攻略
* 一些小功能

如果有任何建議或是想要新增的功能，歡迎在 Issues 中提出。
